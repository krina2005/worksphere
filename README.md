# Workshere

A sleek, mobile-launcher-style dashboard built to help developers and creators discover the best career platforms, remote jobs, and freelancing sites.

## Features
- **App Launcher UI**: A native smartphone-style home screen launcher.
- **Discover Deck**: Interactive swipe/shuffle cards to preview, skip, or save career platforms.
- **Interactive Matchmaker**: A 2-step recommendation quiz tailored to your career profile.
- **Favorites Hub**: Save, clear, and manage your shortlisted platforms.
- **Responsive Layout**: Optimized for both desktop device simulation and full mobile screens.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Getting Started

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the application.
